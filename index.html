Fixing generators around the map is the only way to open the gates and try to escape. Survivors can team up to speed things up while danger gets closer.
When put on a hook, the survivor starts slowly losing health and needs to be rescued by another player within a limited time. If that doesn’t happen, they’re eliminated from the match
The perk system works like a set of abilities that shape each character’s playstyle. Survivors and killers can equip up to four perks before a match, mixing and matching them for different advantages.
The totem mechanic involves objects scattered around the map that can affect the match. Some stay inactive, while others activate special killer abilities, directly influencing the game.
The pallet mechanic involves wooden boards scattered around the map that survivors can drop to block the killer’s path and buy time during chases.
